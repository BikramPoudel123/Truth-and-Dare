## 🚀 How to Get Started

### Step 1️⃣ - Run Setup Script (1 minute)
```bash
# Windows
setup.bat

# macOS/Linux  
bash setup.sh
```

### Step 2️⃣ - Start Backend (Terminal 1)
```bash
python server.py
```

### Step 3️⃣ - Start Frontend (Terminal 2)
```bash
npx expo start
```

### Step 4️⃣ - Play!
- Press `i` → iOS
- Press `a` → Android
- Press `w` → Web
- Scan QR → Expo Go
